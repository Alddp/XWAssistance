### 文件操作

| 命令         | 参数      | 功能                   | 开发进度     |
|------------|---------|----------------------|----------|
| `help`     | 无       | 显示所有                 | `Finish` |
| `show`     | `文件夹路径` | 显示指定文件夹下的所有文件        | `Finish` |
| `format`   | `文件夹路径` | 匹配姓名后格式化命名(正则)       | `Finish` |
| `convetr`  | `文件夹路径` | 将中文大写转换为拉伯数字 (正则)    | `Finish` |
| `simplify` | `文件夹路径` | 解压文件后两层文件夹名字相同，则删除一层 | `Finish` |
